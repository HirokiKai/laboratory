# Gemini Oneword (日本語 → 日本語ひとこと要約)

## 目的
Xの日本語投稿を読みやすくするため、投稿内容を「ひとこと要約」に置き換えて表示する。ユーザーが内容を確認したい場合は「さらに表示」で元の文に切り替えられる。

## 対象
- 対象サイト: X.com / twitter.com
- 対象言語: 日本語投稿
- 対象要素: `data-testid="tweetText"`

## 表示仕様
- 初期表示: 「ひとこと要約」 + 「さらに表示」
- 「さらに表示」を押すと元の文に切り替え
- もう一度押すと「ひとこと要約」に戻す（トグル）
- 視認性のため要約は薄いアクセント色を使用

### 例
```
すっごい、長文。難しい文章
↓
{ひとこと}
さらに表示
```

## 要約仕様
- 日本語 → 日本語の要約
- ひとことの目安: 15〜25文字
- 出力は「要約のみ」。前置き/解説/記号は付けない

## 翻訳（要約）エンジン
- Gemini API を使用
- prompt例:
  - 「次の日本語を15〜25文字の“ひとこと”に要約してください。出力は要約だけ。」
- 複数投稿はバッチ処理し、区切りは `---SEPARATOR---`

## 期待動作
- 日本語投稿は置き換えられる
- 日本語以外は変更しない
- 失敗時は元の文のまま
- 同じ投稿は再処理しない

## 状態管理
- 要約済み: `data-gemini-oneword="true"`
- 表示モード: `data-gemini-oneword-mode="summary|original"`

## UI/UX
- 省略リンク文言: 「さらに表示」
- 展開リンク文言: 「元に戻す」
- クリックターゲットは要約の末尾に小さく表示
